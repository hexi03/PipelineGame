## Project Structure

The project structure is organized as follows:
```
Assets/
|-- Scripts/                  # C# game scripts
|   |-- GameManager.cs        # Central game control script
|   |-- Level.cs              # Handles level configurations
|   |-- MainMenuController.cs # UI main menu control
|   |-- Rocket.cs             # Projectile behavior
|   |-- DebugLogObject.cs     # Debugging helper script
|
|-- StreamingAssets/          # Configuration and JSON files
    |-- levels.json           # Defines game levels
    |-- level1.json           # Specific level configuration
    |-- google-services.json  # Google services config (if applicable)
```
